MY WORKS

Test Plan

2022 Testing Documentation

The Test Plan is a document that describes the entire scope of testing work, starting with the description of the tested objects, strategy, schedule, criteria for the start and end of testing, to the necessary equipment, special knowledge, as well as risk assessment with options for their permission.

Test Case

2022 Testing Documentation

A Test Case is a document that describes a set of steps, specific conditions and parameters necessary to test the implementation of a function or part of it.

Test Suite

2022 Testing Documentation

Test Set is a set of test cases that are united by the fact that they relate to one tested module, functionality, priority or one type of testing.

Check List

2022 Testing Documentation

A Checklist is a document that describes what should be tested. It allows you not to forget about important points, record the results of your work and track statistics about the status of the software product.

Bug Report

2022 Testing Documentation

The report on finding the error is a detailed step-by-step instruction for reproducing the error.
